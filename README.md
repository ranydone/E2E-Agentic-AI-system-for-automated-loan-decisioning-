# Agentic Loan Origination System

An end-to-end Agentic AI backend demonstrating automated loan decisioning.

## Features
- ML-based credit scoring
- Rule-based policy checks
- Agentic orchestration

## Run
1. Install deps: pip install -r requirements.txt
2. Train model: python models/train_model.py
3. Start API: uvicorn app:app --reload
